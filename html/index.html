Presentation

Introduction

Ce Programme affiche une grille a l'ecran une sur laquelle se deplace un joueur Le jeu se termine si le joueur meurt ou si il se deplace sur la case verte en ayant auparavant tuer tous les monstres presents sur la grille

Commandes

Lancement

Fleche du haut : Lancer le jeu Fleche du bas : lancer l'editeur Esc ou Q : Quitter

Jeu

Le deplacement du personnage se fait a l'aide des touches flechees R : redemarre la partie Q ou ESC : Retour au menu principal

Editeur

L'editeur fonctionne de la maniere suivant : un clic sur un sprite de la toolboox charge ce dernier
Un clic gauche sur la grille place le sprite a la case sur la quelle le joueur a clique
Un clic droit sur la grille supprime le sprite place la ou le joueur a clique
C : charge la grille contenue dans grille.txt
N : Affiche une nouvelle grille vide
S : sauvegarde la grille dans test.txt
M : Affiche le menu de l'editeur
Q : Quitte l'editeur
Contraintes : On ne peut pas placer des salles tournantes les unes a cote des autres
On ne peut pas placer plusieurs joueur sur la grille
On ne peut pas placer de salles tournantes sur les bords de la grille
